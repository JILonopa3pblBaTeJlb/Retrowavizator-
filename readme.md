# 🎹 Synthwave Video Production Suite

Набор Python-инструментов профессионального уровня для автоматического монтажа видео под музыку и генерации процедурной Synthwave-графики на GPU. Оптимизировано для чипов Apple Silicon (M1/M2/M3).

## 🚀 Основные возможности
*   **Auto-Montage (`montaj.py`):** Анализирует аудио-транзиенты (бочку/снейр) и автоматически шинкует футажи из папки точно в бит. Включает эффект "Pulse Zoom", привязанный к RMS звука.
*   **Music Visualizer (`clipmaker.py`):** Процедурная 3D-сцена на шейдерах (OpenGL 3.3). Рендерит бесконечную сетку, ретро-солнце и визуализирует ваши `.obj` модели в такт музыке.
*   **HQ Looper (`gifmaker.py`):** Создает идеально зацикленные видео-лупы для Telegram/Instagram с максимальным качеством и минимальным весом.

## 🛠 Установка

1. **Системные зависимости:**
   Установите [FFmpeg](https://ffmpeg.org/) (необходим для сборки видео с аудио).
   ```bash
   brew install ffmpeg
   ```

2. **Python окружение:**
   Рекомендуется Python 3.11+.
   ```bash
   pip install pygame moderngl numpy librosa imageio moviepy opencv-python
   ```

## 📂 Подготовка файлов

Перед запуском структура папок должна выглядеть так:
*   `montaj.py`, `synthwave.py`, `gifmaker.py` — скрипты.
*   `track.mp3` — ваш аудиофайл (любой `.mp3`, `.wav`, `.m4a`) положите в корень.
*   `footage_folder/` — папка с исходниками видео для монтажа.
*   `obj/` — папка с вашими 3D моделями в формате `.obj`.

## 🎮 Инструкция по применению

### 1. Автоматический бит-монтаж (`montaj.py`)
Скрипт найдет музыку, просканирует футажи и соберет динамичный клип.
```bash
python montaj.py
```
*   **Как это работает:** Алгоритм `onset_detect` ищет пики перкуссии. Каждая склейка видео происходит строго на удар. RMS-огибающая управляет зумом кадра в реальном времени.

### 2. Synthwave Визуализатор (`clipmaker.py`)
Запускает интерактивное окно с графикой.
```bash
python clipmaker.py
```
**Управление:**
*   `Space`: Сменить 3D модель на следующую из папки `obj/`.
*   `V`: Включить/выключить авто-вращение модели.
*   `J`: Режим "Shake" (модель вибрирует в такт басу).
*   `Стрелки / WASD`: Перемещение модели в пространстве.
*   `Колесо мыши`: Масштабирование.
*   `Enter`: **Начать HQ Рендеринг**. Скрипт просчитает видео покадрово и склеит его с аудио через FFmpeg.

### 3. Создание цикличных лупов (`gifmaker.py`)
Используется для создания "вечных" анимаций.
```bash
python gifmaker.py
```
*   Сгенерирует файл `telegram_hq_loop.mp4`. Оптимизировано под плавное зацикливание сетки и вращение модели ровно за 4 секунды.

## ⚙️ Технические детали (для разработчиков)
*   **Рендеринг:** Используется `ModernGL` для прямого доступа к буферам OpenGL.
*   **Аудио:** `librosa` используется для быстрого преобразования Фурье и выделения перкуссионной составляющей отдельно от гармонической.
*   **Оптимизация M2:** В `montaj.py` установлено `threads=13` для максимальной загрузки всех ядер процессора при кодировании видео.
